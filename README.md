# Crop Yield Prediction Machine Learning Project

This project was completed for my Machine Learning module at the University of Sussex. It demonstrates an end-to-end machine learning pipeline, beginning with a set of disparate source tables containing missing values and inconsistent structures, and progressing through data cleaning, feature preparation, imputation, model training, prediction, and evaluation.

The project includes the full workflow used to prepare the data, select and implement an imputation strategy, train the predictive model, run an example forecast, and evaluate the model’s performance in the accompanying report.
